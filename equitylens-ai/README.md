# EquityLens AI

A professional Next.js App Router MVP for an AI-powered U.S. stock research platform for English-speaking individual investors.

## Features

- Homepage with fintech SaaS positioning
- Stock search page with mock stock data
- Dynamic stock detail pages
- Stocks to Watch page
- Earnings page
- Daily Market Brief page
- Watchlist page
- Pricing page
- About page
- Educational disclaimer on all pages
- Clean component structure ready for OpenAI API, stock data APIs, authentication, and Stripe

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Mock data files

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Important Product Boundary

EquityLens AI is designed as an educational stock research platform. It avoids direct investment advice language and does not provide personalized investment recommendations.
